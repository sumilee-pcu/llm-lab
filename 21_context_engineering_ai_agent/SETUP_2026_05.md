## Cautions

- `langchain-community` imports currently work, but the package emits a deprecation warning saying it is being sunset. Keep using it only for textbook compatibility, and prefer standalone integration packages for new code.
- The original examples were updated to default to Gemini (`gemini-2.5-flash`) instead of OpenAI.
- `models/embedding-001` was replaced with `models/gemini-embedding-2` after live API verification.
- Examples that call Gemini will fail without a valid `GOOGLE_API_KEY`.
- Local Ollama examples may require unrestricted localhost access in sandboxed terminals.
- Do not paste real API keys into committed files. Use `.env`, which should remain local.
